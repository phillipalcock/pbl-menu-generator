## Install in Cowork

In the Cowork chat, run:
```
/plugin marketplace add https://github.com/phillipalcock/pbl-menu-generator
```

Then install the plugin:
```
/plugin install pbl-menu-generator@pbl-menu-generator
```

## Available Plugins

### pbl-menu-generator
Generates a complete PBL project choice system from a single teacher input file.

**What you get:**
- PowerPoint menu slide (5 columns of project cards, print/display ready)
- 20 student project briefs (one per card, as .docx files)
- Teacher launch guide with coaching questions and rubric template

**How to use:**
1. Fill in `pbl-menu-input.md` with your class details (5 minutes)
2. Run `/pbl-menu-generator:generate` in Cowork
3. Pick up outputs from `/mnt/user-data/outputs/pbl-menu/`
